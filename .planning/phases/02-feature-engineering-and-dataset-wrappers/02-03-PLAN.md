  <action>
Create `scripts/smoke_test_dataset.py`. The script must be runnable from the project root with `python scripts/smoke_test_dataset.py` and must exit 0 if all shape assertions pass, non-zero if any fail.

Script logic:
1. Load `data/processed/cleaned.parquet` with `pd.read_parquet("data/processed/cleaned.parquet")`.
2. Load `data/processed/splits.json` with `json.load(open("data/processed/splits.json"))`.
3. Call `build_samples(df, split, sequence_length=50)` to get sample dicts.
4. Filter each split to `STRICT_DEFENSIVE_POSITIONS` — use a list comprehension:
   `[s for s in samples["train"] if s["position"] in STRICT_DEFENSIVE_POSITIONS]`
5. Build four datasets:
   - `train_a = DefensiveTrajectoryDataset(train_samples, df, sequence_length=25, include_ball_destination=False)`
   - `train_b = DefensiveTrajectoryDataset(train_samples, df, sequence_length=25, include_ball_destination=True)`
   - `val_a   = DefensiveTrajectoryDataset(val_samples,   df, sequence_length=25, include_ball_destination=False)`
6. Build two DataLoaders (num_workers=0, batch_size=64, shuffle=False):
   - `loader_a = DataLoader(train_a, batch_size=64, shuffle=False, num_workers=0)`
   - `loader_b = DataLoader(train_b, batch_size=64, shuffle=False, num_workers=0)`
7. Fetch one batch from each loader with `next(iter(loader))`.
8. Run assertions and print a formatted report:

```
=== Phase 2 Smoke Test ===

Dataset sizes:
  train_a: {len(train_a)} samples  (expected: 52779)
  train_b: {len(train_b)} samples  (expected: 52779)
  val_a:   {len(val_a)} samples    (expected: 7497)

Batch shapes:
  Model A: input={batch_a['input'].shape}  padding_mask={batch_a['padding_mask'].shape}  target_xy={batch_a['target_xy'].shape}
  Model B: input={batch_b['input'].shape}  padding_mask={batch_b['padding_mask'].shape}  target_xy={batch_b['target_xy'].shape}

Assertions:
  [PASS/FAIL] Model A input shape == (64, 25, 50): {batch_a['input'].shape == torch.Size([64, 25, 50])}
  [PASS/FAIL] Model B input shape == (64, 25, 52): {batch_b['input'].shape == torch.Size([64, 25, 52])}
  [PASS/FAIL] Model A no ball cols: {batch_a['input'].shape[-1] == 50}
  [PASS/FAIL] Model B last 2 cols non-zero: {batch_b['input'][:, :, 50:].abs().sum().item() > 0}
  [PASS/FAIL] train_a size == 52779: {len(train_a) == 52779}

=== ALL ASSERTIONS PASS ===   (or lists failures and exits 1)
```

If any assertion fails, print the failure and call `sys.exit(1)`.

Note on batch_size: if `len(train_a) < 64` (won't happen on real data) adjust batch size down. For real data this is not a concern.

Note on memory: building the context index for 4.88M rows takes ~30-60 seconds and ~2-3GB RAM. The script should print a progress message before `DefensiveTrajectoryDataset(...)` is called so the user knows it is working:
```python
print("Building context index for train_a (this takes ~30-60s)...")
```
