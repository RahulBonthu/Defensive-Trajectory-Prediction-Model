  <behavior>
Module-level constants (hard-coded, not configurable):
  - STRICT_DEFENSIVE_POSITIONS = {"CB", "FS", "SS", "LB"}
  - CONTEXT_PLAYERS = 21
  - CONTEXT_FEATURES_PER_PLAYER = 2   (x, y only)
  - SEQUENCE_LENGTH = 25              (default T)

__init__(samples, context_df, sequence_length=25, include_ball_destination=False):
  - Filter `samples` to only those with position in STRICT_DEFENSIVE_POSITIONS
  - Store filtered list as self.samples
  - Build context index from context_df (pre-computed dict, not live DataFrame scan)
  - Set self.context_df = None after index is built (prevent OOM with DataLoader workers)

_build_context_index(context_df):
  - Build {(gameId, playId): {frameId: {nflId: np.array([x, y])}}} from context_df
  - Use only columns: gameId, playId, frameId, nflId, x, y
  - Also build {(gameId, playId): np.array([ball_land_x, ball_land_y])} for ball destinations
    (read from ball_land_x / ball_land_y columns if present; fallback: use sample["ball_target_xy"])

__len__() -> int:
  - Returns len(self.samples)

__getitem__(idx) -> dict:
  - Load sample = self.samples[idx]
  - Slice frames to self.seq_len: own_kin = sample["frames"][:self.seq_len]  shape (T, 8)
  - Slice mask: mask = sample["padding_mask"][:self.seq_len]                  shape (T,) bool
  - Assemble social context: ctx = _assemble_social_context(...)              shape (T, 42)
  - Zero-out social context at padded timesteps: ctx[~mask] = 0.0
  - Concat: tensor = np.concatenate([own_kin, ctx], axis=-1)                  shape (T, 50)
  - If include_ball_destination:
      ball_xy = _get_ball_destination(sample)                                  shape (2,)
      ball_tile = np.tile(ball_xy, (self.seq_len, 1))                          shape (T, 2)
      tensor = np.concatenate([tensor, ball_tile], axis=-1)                    shape (T, 52)
  - Return:
      {
        "input": torch.tensor(tensor, dtype=torch.float32),        # (T, 50) or (T, 52)
        "padding_mask": torch.tensor(mask, dtype=torch.bool),      # (T,)
        "target_xy": torch.tensor(sample["target_xy"], dtype=torch.float32),  # (2,)
        "position": sample["position"],
      }

_assemble_social_context(game_id, play_id, frame_ids, target_nfl_id) -> np.ndarray (T, 42):
  - For each frame_id: get all other players' (x, y), sort by nflId ascending
  - Exclude target_nfl_id from context slots
  - Pad/truncate to exactly CONTEXT_PLAYERS=21 slots
  - Missing frame IDs in index return all-zero row (dict.get(fid, {}) handles this)
  - Return shape (T, CONTEXT_PLAYERS * CONTEXT_FEATURES_PER_PLAYER) = (T, 42)

_get_ball_destination(sample) -> np.ndarray (2,):
  - Try self._ball_index[(game_id, play_id)]
  - Fallback: sample["ball_target_xy"]
  - If both None: raise ValueError(f"No ball destination for game {game_id} play {play_id}")
