**Fixture 1: `minimal_samples`**
Returns a list of 6 sample dicts (the same structure `build_samples()` produces) representing 4 CB, 1 FS, and 1 SS player-play pairs — all with strict positions. Use hardcoded values:
- `sequence_length = 25` (T=25)
- `frames`: shape (25, 8) float32 array — first 10 rows non-zero (real frames), rows 10-24 all zeros (padded)
- `padding_mask`: shape (25,) bool — True for indices 0-9, False for 10-24
- `target_xy`: shape (2,) float32, e.g. `[2.5, -1.0]`
- `ball_target_xy`: shape (2,) float32, e.g. `[15.0, 3.0]`
- `gameId`: use 2023010100 for samples 0-3, 2023010200 for samples 4-5
- `playId`: use 1, 2, 3, 4, 1, 2
- `nflId`: use 100, 101, 102, 103, 104, 105
- `position`: "CB" for indices 0-3, "FS" for index 4, "SS" for index 5

**Fixture 2: `minimal_context_df`**
Returns a pandas DataFrame with columns: `gameId`, `playId`, `frameId`, `nflId`, `x`, `y`, `position`.
Build 3 players × 10 frames for each of the 2 games above (2 plays each = 4 game-play combos).
Players per play: nflId 100 (CB), 200 (QB — an "other" player providing social context), 201 (WR).
Frame IDs: 1 through 10.
x values: `nflId * 0.01 + frameId * 0.5` (arbitrary, deterministic).
y values: `frameId * 0.2`.
This gives enough other-player rows for social context assembly without disk access.

Example implementation sketch:
```python
@pytest.fixture
def minimal_samples():
    T = 25
    real_frames = 10
    samples = []
    positions = ["CB", "CB", "CB", "CB", "FS", "SS"]
    game_ids  = [2023010100, 2023010100, 2023010100, 2023010100, 2023010200, 2023010200]
    play_ids  = [1, 2, 3, 4, 1, 2]
    nfl_ids   = [100, 101, 102, 103, 104, 105]
    for i in range(6):
        frames = np.zeros((T, 8), dtype=np.float32)
        frames[:real_frames] = np.ones((real_frames, 8), dtype=np.float32) * (i + 1)
        mask = np.zeros(T, dtype=bool)
        mask[:real_frames] = True
        samples.append({
            "gameId": game_ids[i],
            "playId": play_ids[i],
            "nflId": nfl_ids[i],
            "frames": frames,
            "padding_mask": mask,
            "target_xy": np.array([2.5, -1.0], dtype=np.float32),
            "ball_target_xy": np.array([15.0, 3.0], dtype=np.float32),
            "position": positions[i],
        })
    return samples
```
