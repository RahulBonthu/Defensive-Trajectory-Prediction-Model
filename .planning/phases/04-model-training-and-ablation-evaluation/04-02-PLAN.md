Implement `train_one_model(model_variant, train_loader, val_loader, num_epochs=50, lr=1e-3, checkpoint_dir="models", seed=42, wandb_run=None)`:

- Map variant to input_dim: "A" -> 50, "B" -> 52; raise ValueError for other values
- Build config dict with all hyperparameters (d_model=128, nhead=4, num_layers=2, dim_feedforward=256, dropout=0.1, lr, num_epochs, seed, model_variant, input_dim)
- Set seeds: `torch.manual_seed(seed)`, `np.random.seed(seed)`
- Set MPS num_workers: if device is MPS, set num_workers=0 (already handled by caller in DataLoader construction, but document this)
- Instantiate model with confirmed hyperparameters, move to device
- Optimizer: Adam(model.parameters(), lr=lr)
- Epoch loop with tqdm(range(num_epochs), desc=f"Training Model {model_variant} seed={seed}"):
  - Train phase: model.train(); accumulate batch RMSE losses using rmse_loss(); compute epoch mean
  - Val phase: model.eval(); torch.no_grad(); accumulate val losses
  - wandb logging: if wandb_run is not None: wandb_run.log({...}); else: wandb.log({...})
  - Best-val checkpointing: if val_loss < best_val_loss: torch.save(state_dict, checkpoint_path)
- checkpoint_path = Path(checkpoint_dir) / f"model_{model_variant.lower()}_seed{seed}_best.pt"
- Path(checkpoint_dir).mkdir(parents=True, exist_ok=True)
- Return the result dict as specified in the interface contract above
