### preprocessor.py contract
```python
def normalize_coordinates(df: pd.DataFrame) -> pd.DataFrame:
    """
    1. Compute los_field_x = absoluteYardlineNumber + 10  (CRITICAL: end-zone offset)
    2. For playDirection == "left":
       a. Mirror x: df.loc[left_mask, "x"] = 120 - df.loc[left_mask, "x"]
       b. Mirror y: df.loc[left_mask, "y"] = 53.3 - df.loc[left_mask, "y"]
       c. Flip angles BEFORE encoding: df.loc[left_mask, "dir"] = (df.loc[left_mask, "dir"] + 180) % 360
       d. Same for "o" column
       e. Mirror los_field_x: df.loc[left_mask, "los_x"] = 120 - df.loc[left_mask, "los_x"]
    3. Subtract los_x from x (LOS-relative)
    4. Center y: df["y"] = df["y"] - 26.65
    NOTE: Use pandas 3.0 CoW-safe pattern — df = df.copy() at start, use .loc for mutation
    """

def encode_angles(df: pd.DataFrame) -> pd.DataFrame:
    """Replace dir and o with dir_sin, dir_cos, o_sin, o_cos. Drop raw degree columns.
    Call AFTER normalize_coordinates (angles already flipped for left plays)."""

def interpolate_missing_frames(df: pd.DataFrame, max_gap: int = 3) -> pd.DataFrame:
    """For each (nflId, gameId, playId) group:
    - Detect missing frameIds in range [min_frame, max_frame]
    - If max consecutive gap > max_gap: set too_many_missing=True, skip interpolation
    - Otherwise: reindex to full frame range, interpolate x, y, s, dir_sin, dir_cos
    - Set is_interpolated=True for filled frames
    Uses scipy.interpolate.interp1d per column."""

def compute_acceleration(df: pd.DataFrame, fps: float = 10.0) -> pd.DataFrame:
    """Compute a_computed = diff(s) * fps per (gameId, playId, nflId) group.
    Set NaN on is_interpolated frames. Fill with per-(play, player) mean."""

def make_temporal_split(
    games: pd.DataFrame,
    val_weeks: list[int],
    test_weeks: list[int],
    output_path: Path,
) -> dict:
    """Split games by week into train/val/test. Assert all three sets are disjoint.
    Write JSON with keys train_game_ids, val_game_ids, test_game_ids to output_path."""
```

### sample_builder.py contract
```python
def build_samples(
    df: pd.DataFrame,
    split: dict,
    sequence_length: int = 50,
) -> dict:
    """
    For each (gameId, playId, nflId) group in each split partition:
    - Filter to defensive positions (CB, FS, SS, LB) only — NOTE: this is Phase 1 pre-filtering
      but Phase 2 will enforce it more strictly; for now include all defensive + football rows
    - Construct fixed-length input matrix of shape (sequence_length, num_base_features)
    - Post-pad short sequences with zeros; record padding_mask (True = real, False = padded)
    - Extract ball final position (x, y) at max(frameId) for football rows — store separately
      as ball_target dict keyed by (gameId, playId)
    Returns dict with keys: "train", "val", "test", each a list of sample dicts:
    {
      "gameId": int, "playId": int, "nflId": int,
      "frames": np.ndarray of shape (sequence_length, num_base_features),
      "padding_mask": np.ndarray of shape (sequence_length,) bool,
      "target_xy": np.ndarray of shape (2,),  # ending (x, y) of this player
      "ball_target_xy": np.ndarray of shape (2,) or None,  # ball final position
      "position": str,
    }
    """
```

**src/data/preprocessor.py** — Implement all five functions per the contracts above and the patterns in 01-RESEARCH.md.

Critical implementation order within `normalize_coordinates`:
1. `df = df.copy()` (pandas 3.0 CoW — mandatory first line)
2. Compute `los_field_x = df["absoluteYardlineNumber"] + 10`
3. Create `left_mask = df["playDirection"] == "left"`
4. Apply spatial flip with `.loc`: x → 120 - x, y → 53.3 - y
5. Apply angle flip with `.loc`: dir → (dir + 180) % 360, o → (o + 180) % 360
6. Mirror los_field_x for left plays: los_field_x[left_mask] = 120 - los_field_x[left_mask]
7. Assign `df["los_x"] = los_field_x`
8. Subtract LOS: `df["x"] = df["x"] - df["los_x"]`
9. Center laterally: `df["y"] = df["y"] - 26.65`
10. Return df (do NOT drop los_x — useful for debugging)

For `encode_angles`: apply AFTER coordinate normalization (angles already flipped). Use `np.radians()` + `np.sin()` / `np.cos()`. Drop raw "dir" and "o" columns. Add "dir_sin", "dir_cos", "o_sin", "o_cos".

For `interpolate_missing_frames`: use `groupby(["gameId", "playId", "nflId"]).apply(interpolate_group)`. The inner `interpolate_group` function must:
  - Sort by frameId
  - Find all_frames = range(min, max+1), missing = set difference
  - Check consecutive run lengths in missing (use `np.diff` on sorted missing list)
  - If max run > max_gap: add `too_many_missing=True` column and return early
  - Reindex to full frame range, set `is_interpolated` flag where x was NaN
  - Interpolate x, y, s using scipy.interpolate.interp1d with `kind="linear"`, `bounds_error=False`, `fill_value="extrapolate"`
  - Interpolate dir_sin, dir_cos (after encode_angles has been called — or interpolate on raw dir before encoding; choose the simpler path and document it in a comment)

NOTE: The safest pipeline order is: normalize_coordinates → encode_angles → interpolate_missing_frames → compute_acceleration. Interpolate AFTER encoding so dir_sin/dir_cos (not raw degrees) are interpolated.

For `make_temporal_split`: use `games.loc[~games["week"].isin(val_weeks + test_weeks), "gameId"]` pattern. Convert to sorted lists for JSON serialization. Assert disjointness with descriptive messages.
