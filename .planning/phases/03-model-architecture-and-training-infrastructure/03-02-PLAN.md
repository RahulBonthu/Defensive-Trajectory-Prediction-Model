From 03-RESEARCH.md — critical conventions:
- Conv1d: x.permute(0, 2, 1) BEFORE, x.permute(0, 2, 1) AFTER (restores batch-first)
- padding_mask inversion: src_key_padding_mask = ~padding_mask (DATASET=True:real → TRANSFORMER=True:ignore)
- Mean pool: (x * mask_f).sum(dim=1) / mask_f.sum(dim=1).clamp(min=1)
- RMSE: torch.sqrt(F.mse_loss(pred, target) + 1e-6)
- Attention capture: subclass TransformerEncoderLayer, override _sa_block with **kwargs
- d_model=128 must be divisible by nhead=4
- dropout=0.0 for overfit test (set via constructor arg, not hardcoded)

       ```python
       class AttentionCapturingEncoderLayer(nn.TransformerEncoderLayer):
           def __init__(self, *args, **kwargs):
               super().__init__(*args, **kwargs)
               self.last_attn_weights: torch.Tensor | None = None

           def _sa_block(self, x, attn_mask, key_padding_mask, **kwargs):
               x_out, weights = self.self_attn(
                   x, x, x,
                   attn_mask=attn_mask,
                   key_padding_mask=key_padding_mask,
                   need_weights=True,
                   average_attn_weights=True,
               )
               self.last_attn_weights = weights.detach()  # (batch, T, T)
               return self.dropout1(x_out)
       ```

    4. TrajectoryTransformer(nn.Module)
       Constructor signature:
       ```python
       def __init__(
           self,
           input_dim: int,           # 50 for Model A, 52 for Model B
           d_model: int = 128,
           nhead: int = 4,           # d_model=128 divisible by 4 ✓
           num_layers: int = 2,
           dim_feedforward: int = 256,
           dropout: float = 0.1,
       ) -> None:
       ```
       Layers:
       - self.conv = nn.Conv1d(input_dim, d_model, kernel_size=3, padding=1)
       - encoder_layer = nn.TransformerEncoderLayer(d_model, nhead, dim_feedforward, dropout, batch_first=True)
       - self.encoder = nn.TransformerEncoder(encoder_layer, num_layers=num_layers)
       - self.head = nn.Linear(d_model, 2)

       forward(x: Tensor, padding_mask: Tensor) -> Tensor:
       ```python
       # x: (batch, T, input_dim)
       # padding_mask: (batch, T) bool — True=real, False=padded
       x = x.permute(0, 2, 1)                   # (batch, input_dim, T)
       x = self.conv(x)                           # (batch, d_model, T)
       x = torch.relu(x)
       x = x.permute(0, 2, 1)                   # (batch, T, d_model)
       src_key_padding_mask = ~padding_mask       # INVERT: True=ignore(padded)
       x = self.encoder(x, src_key_padding_mask=src_key_padding_mask)
       mask_f = padding_mask.unsqueeze(-1).float()
       x_pooled = (x * mask_f).sum(dim=1) / mask_f.sum(dim=1).clamp(min=1)
       return self.head(x_pooled)                # (batch, 2)
       ```
